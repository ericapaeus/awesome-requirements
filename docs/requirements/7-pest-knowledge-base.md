# 病虫害知识库

病虫害知识库是一个全面的数字化知识管理系统，旨在为安徽省植保工作者提供专业、权威、易于访问的病虫草害识别和防治知识资源。系统包含病虫草害识别知识库、植保技术知识库和植保员比赛试题库三个主要模块，实现病虫草害知识的积累与分享。
